Accessibility Statement

Last updated: July 6, 2026

Our commitment

MCL Electric LLC is committed to making this website usable by everyone, including people with disabilities. We aim to conform to the Web Content Accessibility Guidelines (WCAG) 2.1, Level AA.

What we've done

  • Semantic HTML structure with proper landmarks, headings, and a skip-to-content link
  • Text and interface colors checked for sufficient contrast
  • Full keyboard operability with visible focus indicators
  • Descriptive alternative text on meaningful images
  • Labels on all form fields and support for browser autofill
  • Animations disabled for visitors who prefer reduced motion
  • A responsive layout that works with screen magnification and mobile screen readers

Found a barrier?

Accessibility is an ongoing effort. If any part of this site is difficult for you to use, please tell us and we'll fix it promptly — and we're always happy to take your service request by phone instead:
